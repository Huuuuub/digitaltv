=================================================
Visual C++ Redistributable for Visual Studio 2017
=================================================

Download page:
  https://www.visualstudio.com/downloads/
  Scroll down to bottom of page, select "Other Tools and Frameworks" and
  then "Microsoft Visual C++ Redistributable for Visual Studio 2017".

Direct links:
  Win32:
    https://go.microsoft.com/fwlink/?LinkId=746571
    https://download.microsoft.com/download/1/f/e/1febbdb2-aded-4e14-9063-39fb17e88444/vc_redist.x86.exe
  Win64:
    https://go.microsoft.com/fwlink/?LinkId=746572
    https://download.microsoft.com/download/3/b/f/3bf6e759-c555-4595-8973-86b7b4312927/vc_redist.x64.exe

Various versions of the Visual C++ Redistributable have slightly different file
names. For consistency, we rename them as vcredist32.exe and vcredist64.exe
so that all installers of TSDuck have the same names.
